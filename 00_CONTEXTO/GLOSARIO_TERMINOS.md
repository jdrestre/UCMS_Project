# GLOSARIO DE TÉRMINOS UCMS (v2.5.0)
**Propósito:** Definición de conceptos y variables con trazabilidad de origen.

| Término | Definición | Origen / Contexto |
| :--- | :--- | :--- |
| **Bloque Único Dinámico** | Modelo de datos plano donde toda la información financiera y técnica viaja en una sola fila. | **GLOBAL (Núcleo)** |
| **Matriz de 65 Columnas** | Estructura específica de celdas diseñada para la complejidad de EPM. | **SPOKE EPM** |
| **Agnosticismo de Salida** | Capacidad del Admin para validar diferentes números de columnas según el servicio. | **GLOBAL (Admin)** |
| **Prefijos (E_, G_, Ac_)** | Identificadores de inicio de columna para separar Energía, Gas, Agua, etc. | **SPOKE EPM** |
| **Smart Alerts** | Etiquetas en mayúsculas (Tags) para identificar anomalías rápidamente. | **GLOBAL / SPOKE** |
| **Costo_Uso_Base** | Valor monetario del consumo o plan base del servicio. | **GLOBAL (Financiero)** |
| **Info_Reg_Json** | Contenedor JSON para texto informativo, decretos y letra pequeña. | **GLOBAL (Observación)** |
| **Factor_Multiplicador** | Valor técnico (Constante) para calcular el consumo real desde lecturas. | **SPOKE (EPM/GAS)** |
